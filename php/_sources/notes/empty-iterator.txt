EmptyIterator
-------------

``EmptyIterator`` is a class provided by PHP that is an implementation of the `Null Object pattern <http://en.wikipedia.org/wiki/Null_object_pattern>`_ for Iterators. It is used in the example code below. ``EmptyIterator::valid()`` always returns false.
